# goimports-reviser

Tool for Golang to sort goimport by 3 groups: std, general and project dependencies

### Use help for usage:
```bash
goimports-reviser -h
```

### Example with bash cmd:
```bash
goimports-reviser -project-name github.com/incu6us/goimport-reviser -file-path ./reviser/reviser.go 
```
